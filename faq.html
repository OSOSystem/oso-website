FAQ

Here is a list of all questions which might arise when you are read or hear about the oso project.


What does OSO mean?

OSO stands for Open Source Ortungssystem. The last word Ortungssystem is German and means locating system.

What are the benefits of using it?

By making use of the technology today we want you too feel safe in situations where usually would not. You can inform multiple family members or friends by simply pushing a button.

How to use it?

Your device should have a sim card slot and be programmable so we can connect it with our service. GPS is also needed for resolving the coordinates to an address but it is not a must have.

How does it work?

When you are device is connected with our service and you are sending an emergency our system will notice it. It tries to identify you, so it can distribute it to the persons which you've listed. Your listed helpers are then getting notified via our app.

Am I being tracked?

As the name OSO already implies every component of this project open source. We only save the data which is needed for the service to work. There always will be transparency.

The future

We are trying to build a platform where people in need have a tool to communicate effortlessly about needing help. We encourage people to work with us so we can provide the best possible experience. There is a lot more to come in the future. To name a few examples

  • Host service which can be used by everyone
  • Support more devices
  • Add Quick responses for help-providers
